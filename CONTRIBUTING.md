# Contributing

The Kubescape project contributing guidelines apply to this repository:
https://github.com/kubescape/project-governance/blob/main/CONTRIBUTING.md

## Requirements for contributions

- **Coding standards**: Code must pass ESLint (`npm run lint`) and TypeScript checks (`npm run tsc`). Run `npm run format` before submitting.
- **Tests**: Run `npm run test` and ensure all tests pass.
- **Sign your commits**: All commits must be signed off (`git commit -s`) in accordance with the [Developer Certificate of Origin](https://developercertificate.org/).
- **Pull requests**: All changes require a pull request and at least one approving review before merging.
